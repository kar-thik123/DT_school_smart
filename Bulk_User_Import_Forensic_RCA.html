11. Proposed / applied fix (smallest)

  1. Ops: Restart backend under nodemon so STEP 9.1 role re-resolution is actually loaded.
  2. Code: Add STEP 9.2 — before createMany, verify every non-null grade_id/section_id against live org-scoped rows; on miss throw Error with statusCode = 400 (same pattern as roles). Normalize empty strings to null.
  3. Parser: Enable csv-parse bom: true.
  4. Analyze grade/section lookup: Case-insensitive name matching via org-wide fetch + uppercase map (prevents false “not found” that previously required brittle exact case).
No changes to RBAC rules, manual user CRUD flow, enrollment, teacher assignment, notifications, or audit write path semantics beyond existing call.
